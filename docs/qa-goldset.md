# 골드셋 질의 · QA 체크리스트 (WBS 1.7)

`PRD.md` §2·§8과 대표 질의를 바탕으로 한 **평가용 초안**이다. 구현 후 자동/수동 회귀에 사용한다.

## 체크리스트 (한 턴 기대)

- [ ] 응답에 **구조화 결과**(서식지·포켓몬·지역·재료)가 도구/DB 근거로 포함된다.
- [ ] **이벤트 서식지**는 `is_event` 또는 설명으로 구분된다.
- [ ] 조건(시간·날씨 등)이 DB에 없으면 **‘정보 없음’** 또는 명시적 null 처리된다.
- [ ] **출처 메타**(`verified_at`, confidence, source 요약)가 노출 가능하다.

## 골드셋 질의 (예시)

| # | 질의 유형 | 질의 (한국어) | 기대 |
| --- | --- | --- | --- |
| G1 | 포켓몬→서식지 | 부우부가 사는 모든 서식지 보여줘 | 해당 포켓몬이 spawn 규칙에 있는 서식지 목록 + 지역·재료 |
| G2 | 서식지→포켓몬 | 은은한 빛과 꿈나라로에는 어떤 포켓몬이 살아? | spawn 규칙 기반 목록, 이벤트 여부 |
| G3 | 지역/필터 | 바싹바싹 황야 마을에서 밤에 유리한 서식지 추천 | 조건 필터 + 근거 필드 |
| G4 | 모호함 | 야부엉 나오는 어두운 침대 서식지 | 후보 엔터티 또는 disambiguation |

추가 행은 데이터가 채워진 뒤 PRD 표준 질의 50개 등으로 확장한다.
